# Gender Equality Data Website

This project was developed for the course **DS819: Data Driven Applications**.

## Project Goal
The website raises awareness about:
- Forced marriage
- Female genital mutilation (FGM)

Related to **UN Sustainable Development Goal 5: Gender Equality**.

## Technologies Used
- HTML
- CSS
- JavaScript
- REST APIs
- JSON data

## APIs Used
- New York Times Article Search API
- Open Library Books API

## Features
- Dynamic infoboxes using DOM manipulation
- API data integration
- Data visualization
- Narrative storytelling structure
